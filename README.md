# 🧠 Тренажёр мышления — Инструкция по деплою

## Шаг 1 — Получи API ключ на Polza.ai

1. Зайди на **polza.ai**
2. Зарегистрируйся → пополни баланс картой РФ или СБП (от 100 ₽)
3. Раздел **API Keys** → нажми "Создать ключ"
4. Скопируй ключ (начинается с `sk-...`) — он нужен на шаге 3

---

## Шаг 2 — Загрузи файлы на GitHub

1. Зайди на **github.com** → войди в аккаунт
2. Открой репозиторий `thinking-trainer`
3. Нажми **"uploading an existing file"**
4. Перетащи все файлы из этой папки:
   - `index.html`
   - `vercel.json`
   - папку `api` с файлом `chat.js`
5. Нажми **"Commit changes"**

---

## Шаг 3 — Задеплой на Timeweb Cloud App Platform

1. Зайди на **timeweb.cloud** → войди в аккаунт
2. В левом меню нажми **"App Platform"**
3. Нажми **"Создать приложение"**
4. Выбери **"GitHub"** → подключи аккаунт → выбери репозиторий `thinking-trainer`
5. Тип приложения: **Node.js**
6. Нажми **"Создать"**

---

## Шаг 4 — Добавь API ключ в переменные окружения

1. В панели Timeweb открой своё приложение
2. Вкладка **"Переменные окружения"**
3. Добавь переменную:
   - **Имя:** `ANTHROPIC_API_KEY`
   - **Значение:** твой ключ от Polza.ai (`sk-...`)
4. Нажми **"Сохранить"** → **"Перезапустить"**

---

## ✅ Готово!

После деплоя Timeweb выдаст ссылку вида `твой-проект.timeweb.app` — её можно сразу отправлять родителям.
